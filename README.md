# 🌦️ Weather App

A simple and responsive **Weather Application** built using **HTML, CSS, and JavaScript**.
This app fetches real-time weather data using a weather API and displays it based on the user’s city input.

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Display real-time temperature
* ☁️ Show weather condition (description)
* ⚡ Fast API-based data fetching
* 📱 Responsive UI

## ⚙️ How It Works

* User enters a city name
* JavaScript sends request to weather API
* API returns weather data
* Data is displayed dynamically on UI

## 📦 Installation & Setup

1. Clone the repository:

```bash id="w6m1z8"

```

2. Open the project folder:

```bash id="u8k3np"
cd weather-app
```

3. Run the app:

* Open `index.html` in your browser

## 🔑 API Setup

1. Get API key from: https://openweathermap.org/api
2. Replace in `script.js`:

```js id="k9p2qs"
const apiKey = "YOUR_API_KEY";
```
